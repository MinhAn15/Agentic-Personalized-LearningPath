# Load Context Workflow

This workflow loads the critical "Memory Bank" files to ensure the AI Agent understands the project's history, goals, and current state.

// turbo-all

1. Read the System Conventions (Rules of Engagement)
   `view_file system_conventions.md`

2. Read the Project Brief (High-level understanding)
   `view_file context/projectbrief.md`

3. Read the Active Context (What we are doing NOW)
   `view_file context/activeContext.md`

4. (Optional) Read System Patterns if architectural decisions are needed
   `view_file context/systemPatterns.md` (If exists)

5. Summary
   - After reading, summarize the current priority task based on `activeContext.md`.
